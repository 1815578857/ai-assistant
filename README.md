# 🎤 AI问答助手

> 语音识别 + 截图识别 + AI 问答 | 手机同步 | 全局快捷键

## ✨ 主要功能

- **🎙️ 语音识别**：实时/标准/本地三种模式，支持麦克风和系统音频
- **📸 截图识别**：全屏或区域截图，AI 自动分析内容
- **🤖 AI 问答**：自动生成答案，支持多模板和上下文对话
- **📱 手机同步**：扫码连接，识别结果和答案实时推送到手机

## 🚀 快速开始

1. 下载：[Releases](https://github.com/1815578857/ai-assistant/releases)
2. 运行 `AI问答助手.exe`

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+R` | 开始/停止录音（可自定义） |
| `Ctrl+Shift+Q` | 截图识别（可自定义） |
| `Ctrl+Shift+T` | 显示/隐藏窗口（可自定义） |
| `Ctrl+S` | 生成答案（可自定义） |

## 📱 手机同步

点击 `📱 二维码` → 扫码连接 → 实时同步

## ❓ 常见问题

- **模型 API 初始化失败**：开启代理 TUN 模式
- **快捷键无效**：以管理员权限运行
- **找不到系统音频**：启用"立体声混音"设备
- **API 配额限制**：15次/分钟，等待几分钟重试
- **手机连接失败**：确保同一 WiFi，检查防火墙
